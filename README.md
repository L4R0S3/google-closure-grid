#google-closure-grid (pear.ui.Grid)

version 0.1 (unstable)

* Demo : http://dekajp.github.io/demos/index.html
* API  : http://dekajp.github.io/docs/ 

##Complete Features

* Data Virtualization ~ 100,000 Rows
* Active Cell and Active Row Highlight
* Sorting
* Column Resizing
* Header Cell Menu
* Paging
* Column Formatting
* Keyboard Navigation
* Data Filter
* Column Move

##Credits
